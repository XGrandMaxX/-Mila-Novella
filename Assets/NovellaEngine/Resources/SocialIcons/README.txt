Положи сюда PNG-иконки соцсетей с прозрачным фоном:

    telegram.png   — для кнопки «Telegram» в окне жалобы
    discord.png    — для кнопки «Discord» в окне жалобы

Требования:
  • Формат: PNG (рекомендуется) или JPG.
  • Прозрачный фон желателен но не обязателен.
  • Размер: 64×64 — 256×256 (квадрат, иконка масштабируется по ScaleToFit).

Когда положишь файл, Unity:
  • автоматически проставит правильные настройки импорта (альфа,
    без сжатия, без MIP) через NovellaSocialIconsImporter.cs;
  • сбросит кэш — иконка появится в окне жалобы без перезапуска Studio.

Если файлов в этой папке нет, окно жалобы рисует свои упрощённые
иконки программно (см. NovellaSocialIcons.BuildTelegram / BuildDiscord).
